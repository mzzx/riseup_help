@title = 'Aliase und Weiterleitung'

h2. Was ist ein E-Mail-Alias?

Ein Alias ist eine andere E-Mailadresse, die an das gleiche Postfach ausgeliefert wird. Du könntest zum Beispiel ein Riseup-Konto [@joe_hill@riseup.@]net haben, aber die neue Adresse [@maydaychicago@riseup.net@] als Kontaktadresse für eine Aktion verwenden. E-Mails, die an diesen Alias gesendet werden, erscheinen dann inm Postfach deines Kontos.

h3. Mit Aliasen Spam verhindern

Wenn du eine E-Mailadresse benötigst, um dich bei eine Internetseite anzumelden, kannst du anstatt deiner normalen Adresse einen Alias erstellen und diesen verwenden. So kannst du auch erfahren, wenn diese Seite beginnt, Spam an dich zu versenden. Falls du das stoppen möchtest, kannst du einfach den Alias löschen, ohne dein E-Mailkonto zu beeinflussen. Raffiniert!

h3. Wie kann ich einen Alias erstellen?

# Melde dich an unter https://account.riseup.net
# Klicke auf *E-Mail*
# Schreibe "ALIASNAME@riseup.net" in das "Alias"-Feld
# Klicke auf *Save*

*Beachte:* Sollte wer anderes diesen Alias oder Benutzernamen verwenden, kannst du dir diesen Alias nicht anlegen. Versuche einen anderen Namen!

h2. Was ist eine E-Mail Weiterleitung?

Wenn du möchtest, dass alle E-Mails, die an deine Riseup E-Mailadresse gesendet werden, in einem Postfach bei einem anderen E-Mailanbieter landen, ist das eine E-Mail-Weiterleitung.

h3. Wie kann ich alle E-Mails an eine andere Adresse weiterleiten?

# Melde dich an unter https://account.riseup.net
# Klicke auf *E-Mail*
# Am Ende der Seite befindet sich ein Feld *Weiterleitung*. Gib dort die Adresse ein, an die deine E-Mails weitergeleitet werden sollen
# Klicke auf *Änderungen speichern*
# *Noch was:* Du könntest ein komisches rotes Ausrufezeichen am oberen Bildschirm sehen. Das ist ein Bug, aber solange die eingetragene Adresse angezeigt wird, werden deine E-Mails weitergeleitet.
